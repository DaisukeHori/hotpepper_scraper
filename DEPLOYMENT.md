## ⚠️ Vercelデプロイ時の注意点

### 1. サーバレス関数の実行時間制限
**Hobby（無料）プラン**: 最大10秒
**Pro プラン**: 最大60秒

**現状**: ローカルテスト（2ページ）で約1秒
**推定**: 10ページで約5秒程度（問題なし）

### 2. 推奨される制限設定

現在の実装では `maxPages` を1-10に制限済み ✅

### 3. 環境変数（必要に応じて）

もしレート制限や遅延を追加する場合：
```env
SCRAPE_DELAY_MS=100
MAX_CONCURRENT_WORKERS=10
```

### 4. デプロイ手順

```bash
# Vercel CLIをインストール（未インストールの場合）
npm install -g vercel

# Vercelにログイン
vercel login

# デプロイ（本番環境）
vercel --prod
```

または、GitHubとVercelを連携（推奨）：
1. https://vercel.com にアクセス
2. GitHubリポジトリをインポート
3. 自動デプロイ設定

### 5. デプロイ後の確認項目

- [ ] `/api/scrape` エンドポイントが動作すること
- [ ] CSVダウンロードが機能すること
- [ ] タイムアウトエラーが発生しないこと
- [ ] エラーログの確認

### 6. パフォーマンスモニタリング
